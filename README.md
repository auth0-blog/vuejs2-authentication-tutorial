## Installation

```bash

# Get the project
git clone git@github.com:auth0-blog/vuejs2-authentication-tutorial.git vuejs2-authentication-tutorial


# Change directory
cd vuejs2-authentication-tutorial

# Install the dependencies
npm install

# Run your app
npm run dev
